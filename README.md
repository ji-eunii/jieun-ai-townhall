# Jieun AI Townhall Report

GitHub Pages용 정적 웹페이지 초안입니다.

## 수정 방법
- 문구 수정: `index.html`에서 텍스트를 직접 수정
- 디자인 수정: `style.css`에서 색상/크기/간격 수정
- 이미지 교체: `assets/` 폴더의 SVG 파일을 같은 파일명으로 PNG/JPG로 교체하거나, `index.html`의 이미지 경로를 수정

## GitHub Pages 배포
1. GitHub에서 새 repository 생성
2. 이 폴더 안의 파일 전체를 업로드
3. Settings > Pages
4. Branch: `main`, Folder: `/root` 선택
5. Save
